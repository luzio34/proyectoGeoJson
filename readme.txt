Versiones de stacks:
Ruby: 3.2.3
Ruby on Rails: 7.1.3.2
Sqlite: 3.45.2
Angular: 17.3.0

Paso a paso para levantar la aplicación:

* Ejecutar la task con el comando: rake 'fetch_earthquake_data:execute'
* Levantar el back con el comando "rails s", se levantará en el puerto 3000.
* Levantar el front con el comando ng serve -o, se levantará en el puerto 4200.
